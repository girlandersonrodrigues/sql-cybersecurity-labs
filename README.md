# SQL Cybersecurity Labs

## About

This repository contains practical SQL investigations focused on cybersecurity, fraud detection, and log analysis.

The goal is to simulate real-world investigations performed by SOC Analysts, Threat Hunters, and Security Analysts using SQL and PostgreSQL.

---

## Objectives

- Investigate authentication logs
- Detect brute-force attacks
- Identify credential spraying attempts
- Analyze firewall events
- Detect suspicious login activity
- Practice SQL for cybersecurity scenarios
- Produce professional investigation reports
